# Calculadora-Gestacional---DPP
Calculadora gestacional que estima a idade da gravidez e a data provável do parto com base na DUM. Desenvolvida com HTML, CSS e JavaScript, possui interface simples, responsiva e intuitiva, aplicando lógica de programação e manipulação do DOM para oferecer uma ferramenta prática na área da saúde.
