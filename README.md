# Calculadora Gestacional-DPP
Calculadora gestacional que estima a idade da gravidez e a data provável do parto com base na DUM. Desenvolvida com HTML, CSS e JavaScript, possui interface simples, responsiva e intuitiva, aplicando lógica de programação e manipulação do DOM para oferecer uma ferramenta prática na área da saúde. Baseada na Regra de Naegele: método padrão utilizado para estimar a Data Provável do Parto (DPP). Ela se baseia no primeiro dia da Data da Última Menstruação (DUM). O cálculo funciona somando dias e ajustando os meses de acordo com o período do ano em que ocorreu a DUM: 

Como calcular a DPP
-> Identifique o 1º dia da última menstruação (DUM).
-> Somar 7 dias ao valor do dia.
-> Ajuste o mês conforme a data da DUM: *De Janeiro a Março: Some 9 meses. *De Abril a Dezembro: Subtraia 3 meses (e some 1 ao ano).
